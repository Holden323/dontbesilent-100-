# dontbesilent 问题消解 100 例

将 dontbesilent 公众号的 100 个问题消解方案打包为一个可安装的 AI Skill。

## 这是什么

当你被模糊的商业、创作、AI 工具、知识付费、自媒体、执行力、对标模仿、产品定价、流量私域类问题困住时，这个 Skill 会用六步消解法把假问题拆成真行动。

覆盖十大领域，100 个高频问题，每个问题按统一六步框架拆解：

1. **原问题** — 用户提出的原始问题
2. **错在哪里** — 指出问题的错误框架
3. **隐含假设** — 暴露问题背后未经验证的预设
4. **大白话改写** — 把模糊问题翻译成可操作的具体问题
5. **真正该做的事** — 给出具体的行动步骤
6. **一句话** — 提炼成可记忆的核心判断

## 安装

在 Claude Code、Cowork 或任何支持 Agent Skills 的工具中说：

```
帮我安装这个 skill：https://github.com/Holden323/problem-dissolution
```

或手动下载 `SKILL.md` 放入你的 skills 目录。

## 使用示例

```
用户：我该不该辞职创业？
Skill 匹配到 091，返回六步消解。
```

```
用户：我的 AI 产品功能很强但没人付费，怎么办？
Skill 没有直接匹配，按六步法现场拆解新问题。
```
